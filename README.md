Personal portfolio site. Plain HTML/CSS, no frameworks.

## Stack

- HTML/CSS with CSS custom properties for light/dark theme
- Instrument Serif + IBM Plex Sans (Google Fonts)
- 10 lines of vanilla JS for theme toggle
- Hosted on Vercel

## Deploy

Push to `main`. Vercel picks it up automatically.
